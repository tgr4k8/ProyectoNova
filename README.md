# ProyectoNova
Aplicación web para academia de baile NOVACLUB. Gestor de contenido ERP creado por mi para el sitio con posibilidad de añadir, clases, estilos, grupos y eventos. Diferentes tipos de usuarios gestionados por JWT, diferenciando entre: Suscrito, Invitado, Profesor y Administrador diferenciando entre funcionalidades por roles.
